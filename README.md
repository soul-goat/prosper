📈 基金助手（Fund Assistant）

一款基于 PyQt5 的桌面基金监控工具，支持 极简模式 / 完整模式 双界面，适合长期后台运行，实时跟踪自选基金的收益情况。

✨ 主要特性一览

极简版 / 完整版双模式自由切换

实时基金预估净值与收益计算

清晰的盈亏颜色与图标提示

本地数据持久化，重启不丢数据

窗口可自由拉伸，适配大屏与高 DPI

🖥 界面模式说明
1️⃣ 极简模式（默认启动）

适合 日常盯盘 / 后台常驻 使用。

功能包括：

始终置顶显示

显示自选基金列表

单只基金信息：

基金名称（过长自动截断）

今日涨跌幅

今日收益金额

收益颜色区分：

红色：正收益

绿色：负收益

底部汇总：

今日总收益（金额）

累计总收益（金额）

支持操作：

手动刷新

定时自动刷新

隐藏到屏幕边缘

悬浮按钮快速恢复

一键切换到完整版

2️⃣ 完整模式

适合 基金管理 / 数据查看 / 操作配置 使用。

功能包括：

🔍 基金搜索与添加

输入基金代码搜索

显示搜索结果：

基金名称

昨日净值

预估净值

今日涨跌幅

自动填充成本价（昨日净值）

支持两种买入方式：

输入买入金额 → 自动计算份额

输入买入份额 → 自动计算金额

添加基金到自选列表

自动防止重复添加

📊 自选基金表格

表格字段包括：

基金代码

基金名称

成本价

持有份额

预估净值

今日涨幅

当前收益

累计收益

操作（删除）

表格特性：

内容只读，防止误操作

自动刷新数据

收益颜色区分（红涨绿跌）

支持窗口拉伸自适应显示

📈 汇总统计

今日汇总：

今日总收益金额

今日收益率

累计汇总：

累计总收益金额

累计收益率

使用天气图标（晴 / 云 / 雨）直观表示收益状态

🪟 窗口与交互设计

支持窗口自由拖动

支持水平 / 垂直拉伸

最大拉伸宽度 = 当前屏幕宽度

不再限制固定最大宽度（如 600px）

600px 宽度仅作为：

“极简模式 → 完整模式” 的自动切换阈值

支持窗口吸附到屏幕左右边缘

隐藏后显示悬浮按钮，点击可恢复窗口

💾 数据存储说明

自选基金数据存储在本地文件：

watchlist.json


程序重启后数据自动恢复

不依赖数据库，轻量稳定

🔄 数据刷新机制

默认每 30 秒 自动刷新一次

支持手动立即刷新

网络异常时：

不会导致程序崩溃

自动跳过失败请求并继续运行

🧩 技术栈

Python 3.x

PyQt5

requests

JSON 本地存储

⚠️ 使用说明

确保已安装依赖：

pip install pyqt5 requests


运行主程序：

python fund_assistant.py


首次使用请在 完整版 中添加基金
